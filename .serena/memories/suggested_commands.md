# Suggested Commands
- Sync deps: `flutter pub get`
- Format sources: `dart format lib test`
- Static analysis: `flutter analyze`
- Run tests (unit/widget): `flutter test`
- Coverage check (manual guideline): `flutter test --coverage`
- Launch app on device/emulator: `flutter run`
- Build release artifacts: `flutter build apk` / `flutter build ios` (per target)
- Update packages: `flutter pub upgrade --major-versions` (manual when needed)
