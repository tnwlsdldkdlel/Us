# 작업 개요
AGENTS.md와 AGENTS_KO.md에 사용자 제공 필수 지침을 반영하고, 향후 작업 시 준수해야 할 항목을 문서화한다.

# 작업 계획
1. 기존 지침 문서를 검토하고 새 요구 사항을 반영할 위치를 결정한다.
2. 두 문서에 '반드시 지켜야 하는 것'과 '필수 가이드 라인' 섹션을 추가 및 업데이트한다.
3. 변경 내용을 검토하고 저장한다.

# 작업 과정
- 기존 AGENTS.md, AGENTS_KO.md 내용을 확인하며 섹션 구조와 분량을 파악함.
- 사용자 지시를 반영한 새로운 두 섹션을 설계하고 전체 문장을 간결하게 재작성함.
- 동일 지침을 한국어 문서에 현지화하여 반영함.
- 계획 및 작업 로그 파일을 __prompts 디렉토리에 작성함.

# 작업 실행 결과
- AGENTS.md, AGENTS_KO.md가 최신 강제 준수 항목과 워크플로우 절차를 포함하도록 업데이트됨.
- __prompts/_plan 및 __prompts/_task에 관련 기록이 생성됨.
